iVinyl

Deze applicatie is ontwikkeld voor het vak CSWPI vak

iVinyl is een applicatie waarin in je een verzameling van vinyl platen kan bijhouden. Je kan als gebruiker een verzameling aanmaken waaraan je platen kunt toevoegen en verwijderen. Daarnaast kun je ook de verzameling verwijderen of aanpassen om eventuele info te veranderen.

ERD

Entiteiten

Label

  • naam: string
  • oprichting: DateTime
  • land: string
  • genre: Genre

Artiest

  • naam: string
  • leeftijd: int
  • land: string
  • bio: string

Genre

  • House, Trance, Rock, Pop

Gebruiker

  • gebruikersnaam: string
  • wachtwoord: number
  • geboorteDatum: string

Verzameling

  • eigenaar: Gebruiker
  • oprichting: DateTime
  • info: string

LP

  • titel: string
  • artiest: Artiest
  • release: int
  • genre: Genre
